## API Chatbot DEEP LEARNING

[Flask](https://github.com/Hoanle396/deep-flask-chatbot) framework python service repository.

## Installation

```bash
$ pip install -r requirements.txt
```

```bash
# development
$ python -m app
```

```bash
$ cd ./models

```
